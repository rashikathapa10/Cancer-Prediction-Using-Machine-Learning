# Cancer Prediction Using Machine Learning

## Project Overview
This project uses Machine Learning techniques to predict whether a breast tumor is malignant (M) or benign (B) based on diagnostic measurements from the Breast Cancer Wisconsin Dataset.

## Dataset Information
- Total Records: 569
- Total Features: 30
- Target Variable (y): Diagnosis
  - M = Malignant
  - B = Benign

## Input Features (X)
The model uses features such as:
- Radius
- Texture
- Perimeter
- Area
- Smoothness
- Compactness
- Concavity
- Symmetry
- Fractal Dimension

## Workflow
1. Data Loading
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training
6. Prediction and Evaluation
